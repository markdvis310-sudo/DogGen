Frequently Asked Questions

How does the dog name generator work by size? +
Select Small, Medium, or Large, then click "Generate Names". The system pulls from curated lists of cute, strong, and funny names tailored for each size — tiny names like "Pip" for small breeds, sturdy names like "Titan" for large dogs.
What is Elimination Mode on the spin wheel? +
Each spin removes the name the pointer lands on. Continue spinning until only one champion name remains — a fun way to narrow down your favorite dog name!
Does the website save my progress? +
Yes! Your current name list, elimination progress, and selected size are saved automatically to your browser's local storage. Close and return later — everything picks up where you left off.
Are these names suitable for any breed? +
Absolutely. The size filter ensures the name fits your dog's physical traits, while the breed-specific section in the menu provides cultural and traditional names for German Shepherds, French Bulldogs, Shibas, and many more.
Why can't I spin the wheel sometimes? +
The spin button is disabled when only 1 name remains (the winner) or if no names exist. Click "Generate Names" for a fresh list or "Reset Game" to restore the current names.
Is my personal data collected? +
No data is sent to any server. We use no tracking cookies, no analytics, and no external scripts beyond fonts. Your privacy is fully protected — see Privacy Policy in the menu.
How can I suggest new names or report issues? +
Use the "Contact & Support" option in the menu. Email supportentry1@gmail.com with your suggestions or feedback — we regularly update name lists based on user input!
